# Mini Search Engine 🔍

A Flask-based mini search engine with:

## 🚀 Features
- Web crawling
- TF-IDF ranking
- Login & Registration
- Two-Factor Authentication (2FA)
- Search history tracking

## 🛠 Tech Stack
- Python (Flask)
- SQLite
- HTML/CSS
- JavaScript

## ▶️ Run Locally
```bash
pip install -r requirements.txt
python app.py
